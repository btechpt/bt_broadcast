Broadcast

Easy send message and pin alert message

Easy send message

You can easily send broadcast messages to the users you want.

We provide several choices of message recipients. This feature will send messages via email and the discus module

  • Email: You can write the email you want
  • Specific User: You can select multiple users who have registered with odoo
  • All User: You can send messages to all users
  • Department: You can send messages to users in the selected department

Easy pin alert message

You can easily create pin alert messages that will appear on all pages.

  • Notification Active?: If this option is activated, banner alerts will appear on all pages
  • Type Notification: Provides color options for banner alerts. The available styles are Warning, Info, Danger
